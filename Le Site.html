    Comment fonctionne le site

    Le site est simple d'utilisation mais contient quelque subtilité, nous allons donc vous expliquer son fonctionnement. Le site utilise un menu dit "Burger" pour vous permettre de naviguer sur le site sans difficulté, il suffit d'appuyer sur le bouton en haut à droite de l'écran pour ouvrir le menu. Ensuite il suffit de séléctionner la page sur laquelle vous voulez vous rendre pour y aller. Quand vous vous retrouver sur une page avec un PDF, il est disponible en lecture mais aussi téléchargable grace au bouton de téléchargement qui ce trouve dans le coin bas droit du PDF en question. Ce bouton ouvrira les PDF dnas un autre onglet, il suffit de lancer le téléchargement de cette nouvelle page. Il est possible de retourner à la page d'acceuil du site en appyant sur le logo en haut à gauche de la page.
